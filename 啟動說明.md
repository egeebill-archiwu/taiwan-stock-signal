# 台股布林訊號系統 — 啟動說明

## 📁 專案結構

```
stock-signal-system/
├── backend/          ← Python FastAPI 後端
├── frontend/         ← Web 前端（可直接瀏覽器開啟）
├── requirements.txt  ← Python 套件清單
└── 啟動說明.md
```

---

## 🖥️ 前端（立即可用，無需 Python）

直接用瀏覽器開啟：
```
stock-signal-system/frontend/index.html
```
前端已內建模擬資料，所有頁面（儀表板、訊號篩選、個股分析、回測、交易紀錄）均可瀏覽。

---

## 🐍 後端（需安裝 Python 3.11+）

### 步驟 1：安裝 Python
下載：https://www.python.org/downloads/
安裝時勾選「Add Python to PATH」

### 步驟 2：安裝依賴套件
```powershell
cd stock-signal-system
python -m pip install -r requirements.txt
```

### 步驟 3：啟動後端 API 伺服器
```powershell
python -m uvicorn backend.main:app --reload --port 8000
```

### 步驟 4：確認 API 正常運行
開啟瀏覽器：http://localhost:8000
API 文件：http://localhost:8000/docs

---

## 📱 手機 PWA 安裝

1. 後端啟動後，在手機瀏覽器開啟 `http://[電腦IP]:8000/frontend/index.html`
2. Chrome 點選「新增至主畫面」即可像 App 使用

---

## ⚙️ 策略參數（backend/config.py）

| 參數 | 預設值 | 說明 |
|------|--------|------|
| BB_PERIOD | 20 | 布林通道週期 |
| BB_STD | 2 | 標準差倍數 |
| CONSECUTIVE_K_THRESHOLD | 3 | 連續 K 棒數量（至少 3 根） |
| MA_PERIOD | 10 | 均線週期 |
| TREND_LOOKBACK | 5 | 趨勢判定回看天數 |

---

## 🔔 Telegram 推播設定

1. 在 Telegram 搜尋 `@BotFather` → `/newbot` → 取得 Bot Token
2. 傳訊給 Bot 後，取得你的 Chat ID
3. 編輯 `backend/config.py`：
   ```python
   TELEGRAM_BOT_TOKEN = "你的BotToken"
   TELEGRAM_CHAT_ID = "你的ChatID"
   ```

| 端點 | 說明 |
